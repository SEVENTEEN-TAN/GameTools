# GameTools

GameTools是一个简单而强大的游戏辅助工具，主要提供自定义准星功能。它允许用户在任何游戏中覆盖显示自定义准星，并支持丰富的个性化设置。

## 功能特点

- **多种准星样式**：提供多种内置准星样式（十字准星、圆形、点状、三角形、方形、菱形等）
- **高度可定制**：
  - 可调整准星大小、透明度和颜色
  - 支持边框粗细调整
  - 支持填充控制（实心/空心）
  - 支持准星描边功能：
    - 可选择描边颜色和透明度
    - 两种描边模式：模糊描边和实线描边
    - 使准星在任何背景下都清晰可见
- **全局热键支持**：通过快捷键轻松控制准星的各种属性
- **系统托盘集成**：最小化到系统托盘，不干扰游戏体验
- **鼠标点击穿透**：准星窗口不会拦截鼠标点击，不影响游戏操作

## 安装说明

1. 从[Releases](https://github.com/Me-turing/GameTools/releases)页面下载最新版本
2. 解压文件到任意目录
3. 运行`GameTools.exe`启动程序

**系统要求**：
- Windows 10/11
- .NET 9.0 或更高版本

## 使用方法

1. 启动应用程序后，准星将默认显示在屏幕中央
2. 使用热键调整准星位置和外观
3. 通过系统托盘图标或Alt+X快捷键打开设置窗口，进一步自定义准星

### 快捷键列表

| 快捷键 | 功能 |
|-------|------|
| Alt+Z | 显示/隐藏准星 |
| Alt+X | 打开设置窗口 |
| Alt+方向键 | 微调准星位置 |
| Alt+Home | 重置准星到屏幕中心 |
| Alt+PageUp | 下一个准星样式 |
| Alt+PageDown | 上一个准星样式 |
| Alt+加号(+) | 增大准星 |
| Alt+减号(-) | 缩小准星 |
| Alt+[ | 降低透明度 |
| Alt+] | 增加透明度 |
| Alt+Q | 退出程序 |

### 设置说明

在设置窗口中，您可以调整以下选项：
- **准星样式**：选择不同的准星形状
- **准星大小**：调整准星的尺寸
- **透明度**：调整准星的透明度
- **边框粗细**：调整准星边框的粗细
- **填充控制**：开启/关闭准星填充
- **准星轮廓**：选择是否使用实线轮廓，调整描边粗细和颜色
- **描边控制**：
  - 启用/禁用描边效果
  - 调整描边透明度
  - 选择描边颜色（黑色、白色、红色、绿色）
- **颜色**：选择准星颜色（红、绿、蓝、黄、品红、青、白、橙）

#### 1. 核心功能设计

  - 获取并显示当前系统上运行的所有进程列表

- **网络限速控制**：
  - 分别控制上行和下行网络速度
  - 速度限制范围：0 KB/s 到自定义最大值
  - 支持预设配置（如：低速、中速、高速）
  - 实时显示当前网络使用情况（带宽监控）

- **快捷键系统**：
  - Alt+N：打开网络控制面板
  - Alt+L：快速启用/禁用当前网络限制

  - 支持自定义快捷键

- **音频反馈**：
  - 网络限制启动时播放提示音
  - 网络限制解除时播放不同的提示音
  - 允许用户自定义提示音或禁用此功能

- **网络限速技术**：

  - 方案D：使用WFP (Windows Filtering Platform) API

- **进程监控**：
  - 使用System.Diagnostics命名空间获取进程信息
  - 使用性能计数器（Performance Counters）监控网络流量
  - 定期刷新进程列表和网络状态

- **UI设计**：
  - 创建单独的NetworkControl窗口
  - 使用WPF数据绑定实现实时更新
  - 提供简洁的滑块控件用于调整网络速度
  - 状态指示器显示当前限速是否生效
